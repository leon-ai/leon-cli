### Prerequisites

- [Node.js](https://nodejs.org/) >= 14
- [npm](https://npmjs.com/) >= 5

### Installation

```sh
# Install Leon CLI
npm install --global https://github.com/leon-ai/leon-cli/releases/latest/download/leon-cli.tar.gz

# Discover all the commands availables
leon --help
```
