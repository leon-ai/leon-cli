### Prerequisites

- [Node.js](https://nodejs.org/) >= 14
- [npm](https://npmjs.com/) >= 7

### Installation

```sh
# Install Leon CLI
npm install --global leon-ai/leon-cli#master

# Check all requirements and install them with permission from user (prompts)
leon create birth
```
